Protocolos de Enrutamiento

Aprende sobre RIP, EIGRP y OSPF de manera interactiva

RIP (Routing Information Protocol)

Tipo: Vector Distancia

Métrica: Número de saltos (hop count)

Máximo: 15 saltos

Actualización: Cada 30 segundos

Ventajas: Simple, fácil configuración

Desventajas: Convergencia lenta, limitado a redes pequeñas

EIGRP (Enhanced Interior Gateway Routing Protocol)

Tipo: Vector Distancia Avanzado

Métrica: Compuesta (BW, delay, load, reliability)

Algoritmo: DUAL (Diffusing Update Algorithm)

Actualización: Solo cuando hay cambios

Ventajas: Convergencia rápida, balanceamiento de carga

Desventajas: Propietario de Cisco

OSPF (Open Shortest Path First)

Tipo: Estado de Enlace

Métrica: Costo (basado en ancho de banda)

Algoritmo: SPF (Shortest Path First) - Dijkstra

Actualización: Solo cuando hay cambios

Ventajas: Escalable, estándar abierto, jerárquico

Desventajas: Complejo de configurar
